概念 蓝钻工程，源于欧洲最先进的施工和管理理念，包括工艺质量保障系统、材料质量保障系统、环境质量保障系统和管理质量保障系统四个有机模块，专注于整个施工过程的系统化，是目前国际上最新的和最具系统性的家庭装修施工和管理标准。

蓝钻工程的意义 蓝钻工程，以向消费者提供蓝钻般的工艺品质和服务品质为核心基点，第一次提出了家庭装修工程要从系统性的高度来整体进行施工和管理，从而克服了其它片面强调材料升级的家装工艺的弊端。

释义 蓝钻般的品质（工程品质和服务品质）
蓝钻工程 蓝钻工程，源于欧洲最先进的施工和管理理念，包括工艺质量保障系统、材料质量保障系统、环境质量保障系统和管理质量保障系统四个有机模块，专注于整个施工过程的系统化，是目前国际上最新的和最具系统性的家庭装修施工和管理标准。VIP服务 业主可由关镇铨设计公司提供的VIP客户服务账号登陆ERF系统，享受VIP服务。通过系统可提前知道施工期间专用客服及施工人员、质量管理员，并可在系统中与客服互动，查询工地现场施工情况。

施工中，工作人员根据公司提供的进度表来控制进度，以免拖延工期，此表也可以在客户VIP服务系统中体现。
